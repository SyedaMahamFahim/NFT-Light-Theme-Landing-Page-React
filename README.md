## Run Locally

Clone the project

```bash
  git clone https://github.com/SyedaMahamFahim/NFT-Light-Theme-Landing-Page-React.git
```

Go to the project directory

```bash
  cd nft-light-theme-landing-page-react
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm  start
```
